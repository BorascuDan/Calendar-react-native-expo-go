# Calendar-react-native-expo-go
Calendar for expo-go
An open-source module that provides fully functional interaction with the device’s main calendar on iOS, Android, and Huawei devices.
It allows applications to seamlessly add events directly to the system calendar while solving common issues present in Expo Calendar.

Key features include:
-> Adds events directly to the device’s primary calendar
-> Prevents duplicate events by checking if an event already exists before inserting it
-> Supports adding multiple events in a single operation
-> Reliable and consistent behavior across iOS, Android, and Huawei
